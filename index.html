CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Two or more elements that will share the same styling can be grouped with a comma.
  • Think of debugging as what code should be doing versus what it is doing.
  • Adding breakpoints will help with debugging.
  • Rule declairs that all elements on the page will have the same properties.
